# Smart OCR Tool 🧠

Upload an image or PDF and extract text using **Google Cloud Vision API**.

Built with:
- Python
- Gradio for GUI
- Google Cloud Vision API for OCR

> ⚠️ Make sure you configure your `smartocr.json` key **locally** (not uploaded).

---

## How to Run Locally 💻

```bash
pip install -r requirements.txt
python app.py
